# UiPath Automation Projects

This repository showcases a collection of **UiPath automation projects** designed to demonstrate real-world **RPA (Robotic Process Automation)** implementations.  
Each project focuses on automating repetitive manual tasks using UiPath Studio — including data scraping, PDF processing, email automation, and OCR.

## Overview

The purpose of this repository is to highlight hands-on UiPath experience through end-to-end process automations.  
Each robot demonstrates workflow design, error handling, reusability, and integration with common business applications like Excel, Outlook, and web portals.

---

##  Project List

| No. | Project Name | Description | Key Components Used |
|-----|---------------|-------------|----------------------|
| 1️⃣ | **PDF Data Scraping Robot** | Extracts invoice and billing information from PDF files and stores the results in Excel format. | Read PDF Text, Regex, Write Range |
| 2️⃣ | **Desktop & Web Data Capturing Robot** | Automates data entry between a desktop application and a web form, synchronizing datasets efficiently. | UI Automation, Excel Application Scope, Dynamic Selectors |
| 3️⃣ | **Real Estate Scraping Robot** | Scrapes property listings (price, location, area) from real estate websites and exports to Excel. | Data Scraping Wizard, Excel Automation |
| 4️⃣ | **Email Automation Robot** | Reads, filters, and processes emails, saving attachments and using custom templates for responses. | Outlook / IMAP Activities, For Each Mail, Save Attachments |
| 5️⃣ | **Optical Character Recognition (OCR) Robot** | Extracts data from scanned receipts and image-based documents using OCR engines. | Document Understanding, OCR Engine, File I/O |
